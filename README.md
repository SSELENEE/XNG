# XNG Research Desk v0.9.6 — NOAA ZIP Upload

Patch:
- NOAA 6-10D / 8-14D support `.zip` upload.
- ZIP is auto-extracted server-side.
- Required files after extraction: `.shp`, `.dbf`, `.shx`.
- Still supports multi-file upload.

Use on iPhone:
1. ZIP all NOAA shapefile sidecars together.
2. Upload the ZIP in NOAA 6-10D or NOAA 8-14D.
3. Click Auto Fill from NOAA Shapefile.
